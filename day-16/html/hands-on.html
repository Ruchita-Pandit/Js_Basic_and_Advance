Prototype & Inheritance Practice

Task 1 : Inspect a Prototype

  1. Create const arr = [1, 2, 3].
  2. Use Object.getPrototypeOf(arr) and log it.
  3. Log: Object.getPrototypeOf(Object.getPrototypeOf(arr))
  4. Log the next prototype step again.
  5. Observe which value ends the prototype chain.
  6. In a comment, draw the complete prototype chain.

Task 3 : Constructor Function Inheritance


                    
    

                        
  1. 
                            Create a constructor:
                            Person(name)
                        
    2. 

                        
  2. 
                            Inside it, set:
                            this.name = name
                        
    3. 

                        
  3. 
                            Add:
                            
                                Person.prototype.greet = function () {
                                console.log("Hi, I'm " + this.name);
                                }
                            
                        
    4. 

                        
  4. 
                            Create another constructor:
                            Student(name, school)
                        
    5. 

                        
  5. 
                            Use:
                            Person.call(this, name)
                        
    6. 

                        
  6. 
                            Set:
                            this.school = school
                        
    7. 

                        
  7. 
                            Link:
                            Student.prototype
                            with:
                            Object.create(Person.prototype)
                        
    8. 

                        
  8. 
                            Add:
                            study()
                            to
                            Student.prototype
                        
    9. 

                        
  9. 
                            Create:
                            new Student("Riya", "IIT Delhi")
                        
    10. 

                        
  10. 
                            Call both:
                            greet()
                            and
                            study()

Bonus Task : hasOwnProperty vs in


                    
    

                        
  1. 
                            Create:
                            
                                const dog = Object.create({ species: "Canis" });
                            
                        
    2. 

                        
  2. 
                            Set:
                            dog.name = "Bruno"
                        
    3. 

                        
  3. 
                            Predict the result of:
                            dog.hasOwnProperty("name")
                        
    4. 

                        
  4. 
                            Predict:
                            dog.hasOwnProperty("species")
                        
    5. 

                        
  5. 
                            Predict:
                            "species" in dog
                        
    6. 

                        
  6. 
                            Predict:
                            "toString" in dog
                        
    7. 

                        
  7. 
                            Run all checks and compare with your prediction.
                        
    8. 

                        
  8. 
                            In a comment, write the one-line rule for when to use:
                            hasOwnProperty
                            vs
                            in
