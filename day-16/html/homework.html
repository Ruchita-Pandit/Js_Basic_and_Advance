Prototype & Inheritance Homework

Task 1 : Multi-Level Prototype Chain

  1. Use Object.create() to build a 3-level chain: tool → vehicle → car.
  2. Add one method at each level.
  3. Confirm that car can call all three methods.
  4. Inspect the prototype chain using Object.getPrototypeOf().
  5. Draw the complete chain in a comment.
