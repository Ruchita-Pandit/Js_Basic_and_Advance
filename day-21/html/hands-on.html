Advanced Array & Object Patterns

Task 1 : Prove Shallow Clone Bug

  1. Create const orig = { name: "Priya", addr: { city: "Jaipur" } }.
  2. Make a shallow copy with spread: const copy = { ...orig }.
  3. Mutate copy.addr.city = "Mumbai". Log orig.addr.city. Note the bug.
  4. Now use structuredClone(orig) and repeat. Confirm orig is untouched.

Task 2 : Immutable Nested Update

                    
    
                        
  1. Take this state: const state = { user: { name: "Priya", prefs: { theme: "light", lang: "en" } } }.
    2. 
                        
  2. Write code that produces a NEW state with prefs.theme set to "dark", leaving state untouched.
    3. 
                        
  3. Use spread, don't mutate.
    4. 
                        
  4. Verify state.user.prefs.theme is still "light".

Task 3 : Update One Item in an Array

                    
    
                        
  1. Take const tasks = [{ id: 1, title: "Learn JS", done: false }, { id: 2, title: "Build app", done: false }].
    2. 
                        
  2. Write a function toggleDone(tasks, id) that returns a NEW array with the matching task's done flipped — others untouched.
    3. 
                        
  3. Call it twice on the same tasks. Verify originals never mutate.
    4. 
                        
  4. Use .map and spread.
    5.

  2. Write a single expression that reads data.user.profile.city and falls back to "Unknown" if anything in the chain is null/undefined.
    3. 
                        
  3. Now another expression reading data.user.profile.bio?.length ?? 0.
