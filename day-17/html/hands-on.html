Task 1 : Basic Class with Getter

  1. Build a Rectangle class with width and height instance properties.
  2. Add a get area() getter that returns width * height.
  3. Add a method scale(factor) that multiplies both dimensions in place.
  4. Create a 2 × 3 rectangle, log the area, scale by 2, then log the area again.
