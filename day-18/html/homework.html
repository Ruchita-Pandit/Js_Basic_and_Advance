Advanced Async & Promise Practice

Task 1 : Build wait(ms)

  1. Write a wait(ms) function.
  2. It should return a Promise.
  3. The Promise resolves after ms milliseconds.
  4. Use it like: wait(500).then(() => console.log("Done"))
  5. Verify the delay works correctly.

Task 3 : Promise.race with Timeout

  2. Use Promise.race().
